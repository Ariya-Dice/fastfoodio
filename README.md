# مستندات پروژه نمونه رستوران (FastFood Sample)

این پروژه یک نمونه کامل جهت مدیریت منو، سبد خرید و سفارش آنلاین برای رستوران/فست‌فود است.
تمام اطلاعات، آدرس‌ها، شماره‌ها و داده‌ها به‌صورت **نمونه** وارد شده‌اند و کاربرد عملی ندارند.

## ساختار اصلی
- **App.tsx**: کامپوننت اصلی و مدیریت تمام منطق پروژه
- **data/mockData.ts**: داده‌های ساختگی شعبه، محصولات و دسته بندی
- **types.ts**: تعریف انواع TypeScript برای ساختارهای داده
- **index.html/index.tsx**: نقطه شروع پروژه و بارگذاری صفحات

## اجرای پروژه
```bash
npm install
npm run dev
```

## کامنت‌گذاری
تمام کدها با توضیح و کامنت حرفه‌ای و کاملاً فارسی مستندسازی شده است.

> توجه: این پروژه صرفاً جهت تست رابط کاربری و منطق Front-End بوده و هرنوع شبیه‌سازی اطلاعات واقعی ممنوع است و کلیه آدرس‌ها/شماره‌ها/تصاویر به صورت صوری وارد شده‌اند.
