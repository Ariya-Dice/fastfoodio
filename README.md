# پروژه رستوران فست‌فودیو (FastFoodio)

پروژه **فست‌فودیو** یک برنامه وب کامل و پیشرفته برای مدیریت منو، سبد خرید و سفارش آنلاین رستوران‌های فست‌فود است. این پروژه در دو نسخه توسعه یافته است:

- **نسخه اصلی**: با استفاده از Tailwind CSS
- **نسخه Bootstrap**: با استفاده از Bootstrap 5

هدف اصلی این پروژه ایجاد نمونه‌ای کامل از برنامه‌های رستوران آنلاین با رویکرد مقایسه دو فریمورک محبوب CSS است.

## هدف پروژه

این پروژه به عنوان یک نمونه کامل از برنامه‌های رستوران‌های آنلاین طراحی شده و شامل تمام ویژگی‌های ضروری یک سیستم سفارش آنلاین می‌باشد. هدف اصلی نمایش قابلیت‌های فنی و طراحی رابط کاربری مدرن است.

پروژه در دو نسخه پیاده‌سازی شده تا امکان مقایسه عملی بین دو فریمورک محبوب CSS فراهم شود و توسعه‌دهندگان بتوانند با توجه به نیاز پروژه، بهترین گزینه را انتخاب کنند.

## نسخه‌های پروژه

### نسخه اصلی (Tailwind CSS)
- استفاده از utility-first CSS framework
- کلاس‌های کوتاه و مستقیم
- سفارشی‌سازی حداکثری
- حجم bundle کمتر

### نسخه Bootstrap
- استفاده از component-based CSS framework
- کامپوننت‌های آماده و تست شده
- توسعه سریع‌تر
- سازگاری گسترده با مرورگرها

هر دو نسخه دارای ویژگی‌های کامل یکسان هستند و تنها در رویکرد طراحی و پیاده‌سازی تفاوت دارند.

## ویژگی‌های کلیدی

### صفحه اصلی (Landing Page)
- **طراحی جذاب**: صفحه اصلی با تصویر پس‌زمینه پویا و انیمیشن‌های زیبا
- **ناوبری ریسپانسیو**: منوی ناوبری که در تمام دستگاه‌ها بهینه نمایش داده می‌شود
- **بخش ویژگی‌ها**: نمایش مزایای رستوران با کارت‌های گرافیکی
- **تم تاریک**: استفاده از تم تاریک برای تجربه کاربری بهتر

### انتخاب شعبه
- **نمایش گرافیکی شعبه‌ها**: هر شعبه با تصویر، آدرس و وضعیت باز/بسته بودن نمایش داده می‌شود
- **انتخاب تعاملی**: امکان انتخاب شعبه با کلیک و انتقال به منوی سفارش
- **اطلاعات کامل**: نمایش زمان تحویل تخمینی برای هر شعبه
- **ریسپانسیو**: نمایش بهینه در موبایل، تبلت و دسکتاپ

### منوی سفارش
- **دسته‌بندی محصولات**: تقسیم محصولات به دسته‌های برگر، پیتزا، پیش‌غذا، نوشیدنی و سالاد
- **جستجوی پیشرفته**: امکان جستجو در میان تمام محصولات
- **سبد خرید هوشمند**: افزودن و حذف محصولات با کنترل تعداد
- **نمایش قیمت**: قیمت‌گذاری شفاف و لحظه‌ای

### سبد خرید
- **پنل کشویی**: سبد خرید به صورت پنل کشویی از سمت راست نمایش داده می‌شود
- **مدیریت اقلام**: امکان تغییر تعداد یا حذف محصولات
- **محاسبه خودکار**: قیمت کل و هزینه ارسال به صورت خودکار محاسبه می‌شود
- **دکمه شناور**: دسترسی سریع به سبد خرید در موبایل

## ساختار فنی پروژه

### تکنولوژی‌های اصلی
- **React 19**: کتابخانه مدرن رابط کاربری
- **TypeScript**: تایپ اسکریپت برای ایمنی کد و توسعه بهتر
- **Tailwind CSS**: فریمورک CSS utility-first برای طراحی سریع
- **Vite**: ابزار ساخت سریع و مدرن
- **Lucide React**: آیکون‌های برداری زیبا

### ساختار فایل‌ها

```
fastfoodio/
├── public/
│   └── images/          # تصاویر محلی محصولات و شعبه‌ها
├── src/
│   ├── App.tsx         # کامپوننت اصلی برنامه
│   ├── data/

│   ├── types.ts        # تعریف انواع TypeScript

│   └── index.html      # فایل HTML اصلی
├── package.json        # تنظیمات npm و وابستگی‌ها
├── tsconfig.json       # تنظیمات TypeScript
├── vite.config.ts      # تنظیمات Vite

```

### کامپوننت‌ها

#### App.tsx (کامپوننت اصلی)
- مدیریت سه وضعیت اصلی برنامه: صفحه اصلی، انتخاب شعبه، منوی سفارش
- کنترل وضعیت سبد خرید و جستجو
- منطق افزودن/حذف محصولات از سبد خرید
- ناوبری بین صفحات مختلف

#### ProductCard (کامپوننت محصول)
- نمایش تصویر، نام، توضیحات و قیمت محصول
- کنترل افزودن به سبد خرید
- نمایش تعداد محصول در سبد خرید
- انیمیشن‌های تعاملی

## طراحی و رابط کاربری

### تم و رنگ‌بندی
- **تم اصلی**: تاریک با زمینه مشکی
- **رنگ اصلی**: زرد (#FCD34D) برای عناصر مهم
- **رنگ ثانویه**: خاکستری‌های مختلف برای متن و پس‌زمینه

### تایپوگرافی
- **فونت اصلی**: VazirMatn (فونت فارسی حرفه‌ای)
- **سایز متن**: از بزرگ تا کوچک برای سلسله مراتب اطلاعات
- **وزن فونت**: استفاده از فونت‌های bold برای عناصر مهم

### ریسپانسیو بودن
- **موبایل اول**: طراحی ابتدا برای موبایل و سپس سایر دستگاه‌ها
- **نقاط شکست**: xs (0px), sm (576px), md (768px), lg (992px)
- **Grid سیستم**: استفاده از CSS Grid و Flexbox

## راه‌اندازی و اجرای پروژه

### پیش‌نیازها
- **Node.js** نسخه ۱۸ یا بالاتر
- **npm** یا **yarn** برای مدیریت بسته‌ها

### مراحل نصب و اجرا

1. **کلون کردن پروژه**:
```bash
git clone <repository-url>
cd fastfoodio
```

2. **نصب وابستگی‌ها**:
```bash
npm install
```

3. **اجرای پروژه در حالت توسعه**:
```bash
npm run dev
```

4. **باز کردن در مرورگر**:
برنامه روی آدرس `http://localhost:3000` اجرا خواهد شد.

### اجرای نسخه Bootstrap

برای اجرای نسخه Bootstrap پروژه:

```bash
cd ../fastfoodio-bootstrap
npm install
npm run dev
```

نسخه Bootstrap نیز روی پورت 3000 اجرا خواهد شد و تمام ویژگی‌های نسخه اصلی را با طراحی Bootstrap ارائه می‌دهد.

### اسکریپت‌های موجود

```json

{
  "scripts": {
    "dev": "vite --host 0.0.0.0 --port 3000",
    "build": "vite build",
    "preview": "vite preview",
    "lint": "eslint . --ext ts,tsx --report-unused-disable-directives --max-warnings 0"
  }
}
```

## توسعه و سفارشی‌سازی

### افزودن محصول جدید
در فایل `data/mockData.ts` محصول جدید را به آرایه `products` اضافه کنید:

```typescript
{
  id: 'unique-id',
  name: 'نام محصول',
  description: 'توضیحات محصول',
  price: 50000,
  image: '/images/product-image.jpg',
  category: 'burgers'
}
```

### افزودن دسته‌بندی جدید
در آرایه `categories` دسته‌بندی جدید اضافه کنید:

```typescript
{
  id: 'new-category',
  name: 'نام دسته‌بندی',
  icon: 'food'
}
```

### سفارشی‌سازی استایل‌ها
استایل‌ها با استفاده از Tailwind CSS قابل تغییر هستند. برای تغییرات عمده‌تر می‌توانید از فایل‌های CSS سفارشی استفاده کنید.
